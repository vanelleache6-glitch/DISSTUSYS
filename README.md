# Distributed Student Information System

## Client–Server Distributed System (TCP Sockets)

### Overview
This project implements a distributed Student Information System using the Client–Server paradigm.
The server stores student records and handles multiple clients using multithreading.

### Technologies Used
- Java
- TCP Socket Programming
- Multithreading
- Synchronized HashMap

### How To Compile
javac *.java

### How To Run
1. Start server:
   java StudentServer

2. Start client (new terminal):
   java StudentClient
